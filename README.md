# 萄萄作业帮助教老师辅助工具

一个基于 Python FastAPI 和 Vue.js 的辅助工具。

## 项目结构
```
├── backend/           # Python FastAPI后端
│   ├── app/          # 应用主目录
│   └── config/       # 配置文件
├── frontend/         # Vue.js前端
│   ├── src/         # 源代码
│   └── public/      # 静态资源
└── docs/            # 项目文档
```

## 技术栈
- 后端：Python FastAPI
- 前端：Vue.js 3 + Vite
- 数据库：MySQL
- UI框架：Element Plus

## 快速开始

### 后端启动
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```
